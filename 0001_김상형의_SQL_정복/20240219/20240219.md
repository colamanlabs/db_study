### 3.3 DB 전용계정 생성

#### 3.3.1 계정생성 
```
20240219_0001.sql

-- 계정생성1
CREATE USER 'studyemp'@'localhost' IDENTIFIED BY 'studyemp';
GRANT ALL PRIVILEGES ON employees.* TO 'studyemp'@'localhost';
FLUSH PRIVILEGES;

-- 계정생성2
CREATE USER 'studyemp'@'%' IDENTIFIED BY 'studyemp';
GRANT ALL PRIVILEGES ON employees.* TO 'studyemp'@'%';
FLUSH PRIVILEGES;
```
